    Assignment AI4

    Assignment Background:

    Timely submission will result in your assignment being returned within 14 days of the submission deadline. Marked Jupyter Notebook manuscripts will be returned with comments from your marker. If you receive a pass or distinction, your marker will attach the model solutions for this assignment via the open pull request on your lesson's GitHub repository.
    The aim of this assignment is to have you engage in a hands-on manner with different sections. Due to the constraint of generating embeddings you will be unable to work with a new dataset.

    Question 1:

    Things move fast in the machine learning world. The ESM1b model was created in 2021, however it was supplanted in the next year by the ESM2 model.

    Head to the card of the ESM2 model on the Hugging Face website. You will see if you scroll down that there are multiple versions of the model, with varying layers and parameters, the largest having 15 billion.

    We would like you to load the esm2_t33_650M_UR50D model into memory, which has comparable parameters to the ESM1b model.

    Question 2:

    Using the ESM2 model and tokeniser you have just downloaded, create a function that takes the following input arguments:

    • a list of protein sequences

    • a list of amino acid positions
    and returns the embeddings of the amino acids at those positions for each sequence.

    For example, given the sequences: ['ACFGPLVIE', 'MAPDEYTWK'] and the positions: [1, 3, 5], your function should return the embeddings of the amino acids at positions 1, 3, and 5 in each sequence.
    • Create a function that generates whole-sequence embeddings and finds the amino acid residue embeddings at positions 24 and 102. Since not all sequences are long enough to have a residue at position 102, your function should only return embeddings for sequences that have a residue at both positions.

    • Store the pair of embeddings in a Python list, appending these to another list for each qualifying sequence.
    Note: the amino acid positions above are 1-based (i.e. position 1 is the first amino acid). Because the ESM tokeniser prepends a special [CLS] token at index 0, the amino acid at 1-based position N sits at tensor index N in last_hidden_state. So no index adjustment is required.

    Question 3:

    A single accuracy score can obscure meaningful patterns in model performance. Some genes may have many correctly classified variants whilst others are consistently misclassified. This could reflect genuine biological complexity, biases in the training data, or limitations of the embedding representation.

    Using the pre-computed test embeddings and the trained model:

    Task 1:

    • The test data has been loaded in for you and the gene IDs extracted using regex, saved as the variable gene_ids

    • Reload the trained model from ./data/best_embedding_model.pt, recreating the MLP_LeakyReLu_binary class and predict function from the classification section

    • Run predictions on the test set using a threshold of 0.19

    • Build a results DataFrame containing: target_id, gene_id, y_true, y_predict
      • Assign it to a variable names results

      • Create a new column which finds misclassified predictions
    Task 2:

    Group by gene_id and calculate, for each gene: total variants, number misclassified, and misclassification rate

    To find the count and sum use pandas .groupby() which will find summary statistics or functions to data in the same category (see below):

    # Per-gene summary
per_gene = (
    results.groupby('gene_id')
    .agg(
        total_variants   = ('misclassified', 'count'),
        n_misclassified  = ('misclassified', 'sum')
    )
    .reset_index()
)


    After, calculate the misclassification rate from the count ("total_variants") and sum ("n_misclassified")

    Display the 10 genes with the highest misclassification rates

    Task 3:

    Produce a bar chart of misclassification rates:
    • Restrict the data to only genes with at least 5 variants in the test set

    • Plot the misclassification rate column

    • Plot a dashed, grey, line at 50% to indicate genes with a higher error rate

    • Give the plot appropriate axis labels and title
